User Stories: 

As a creative entrepreneur I want the ability to make fast, easy, and customizable boxes so that I can sell them on my website. 

As a customer, I want to be able to personalize and customize a box as a gift so that it feels unique to the person that I am giving it to. 

As a user, I want a easily navigatable and clean UI so that building a box will be simple and intuitive. 

As a buisness owner, I want a software that allows me to create a box quickly and efficiently so that my customers keep coming back.
